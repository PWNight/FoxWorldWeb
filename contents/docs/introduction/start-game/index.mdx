## 1. Зайдите на наш Discord сервер
На данный момент верификация новых игроков проходит на нашем [Discord сервере](https://discord.gg/2yyeWQ5unZ).
## 2. Подайте заявку
Зайдите в канал #начать-играть, нажмите на кнопку "Отправить заявку" и заполните появившуюся форму.

Обратите внимание, что мы отказываем в верификации, если ваша заявка нарушает [правила категории 1](https://foxworld.ru/docs/rules)

## 3. Привяжите аккаунт в Minecraft
Зайдите на сервер, зарегистрируйтесь и направляйтесь к порталу перед вами.

Далее вам надо вернуться в Discord сервер, найти в списке участников справа бота "FoxWorld Link" и отправить в личные сообщения код, который вы получили в игре.

## 4. Готово, теперь вы часть нашего сообщества :)
Мы поддерживаем заход со следующих версий:
* Java Edition: 1.20 - 1.20.6
* Bedrock Edition: от 1.20.40 до 1.20.81

Для входа с Java Edition используйте mc.foxworld.ru / 135.181.126.170
Для входа с Bedrock Edition используйте be.foxworld.ru / 135.181.126.170 и порт 25630

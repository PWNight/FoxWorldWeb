Чудо свершилось! Мы с радостью представляем вам первый БЕТА релиз нашего сайта, который приносит огромное количество изменений, исправлений и нововведений
## Нововведения:
* Добавлена карта сервера.
* Добавлены заглушки на страницы, которые находятся в разработке.
* Добавлен вывод гильдий игрока в /me/guilds, а также заглушка с ссылками на поиск и создание гильдии, если гильдий нету.
* Добавлена дата последнего изменения на страницах Вики.
* Добавлено уведомление об устаревших разделах на Вики.
* Кнопка входа в ЛК получила обновление: теперь там отображаются различные состояния игрока (забанен, нет доступа, есть подписка и т.п.), а также появились иконки.
* Разработано API для просмотра и управления гильдиями.
* Добавлена страница создания гильдии и страница просмотра гильдий, в которых вы состоите, в личном кабинете.
* Релиз страниц управления гильдией и её участниками перенёс на v1.0.0-beta2
## Изменения:
* Большинство API, требующих данных о пользователе были переписаны. Теперь они получают в запросе токен пользователя и с помощью его получают всю необходимую информацию, не взаимодействуя с БД с логинами и паролями.
* Заметки в документации были улучшены: теперь вместе с заголовком заметки появилась соответствующая иконка.
* Улучшена мобильная адаптация: добавлены отступы от края экрана и исправлены некоторые ошибки с адаптацией.
* Разработана новая логика для проверки вводимых игроками данных, более быстрая и менее громоздкая.
## Исправления:
* Добавлены заголовки на страницах, где они отсутствовали.
* Исправлено отображение блока статистики.
* Исправлены ссылки на правила, где они имелись.
* Исправлены огромное количество мелких визуальных багов на страницах сайта.
* Исправлено огромное количество технических ошибок/предупреждений в коде сайта.
* Убраны все устаревшие пути API сайта.
* Исправлена ошибка с небезопасным подключением при просмотре карты.
* Исправлена работа всего API сайта.
* Исправлено отсутствие изображений в новостях и вики.
* Исправлено отображение блока новостей на мобильных устройствах.
* Исправлено отображение пароля в форме входа.
* Исправлен критический баг невозможности авторизоваться на мобильных устройствах.

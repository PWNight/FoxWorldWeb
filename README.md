PET проект, разрабатываемый с 2021 по 2025 год.
FoxWorld - Minecraft сервер "ванильного" типа с акцентом на качество и разнообразие.

Сайт содержит в себе:
* Википедию, ака система документации с возможностью редактирования в [отдельном репозитории](https://github.com/PWNight/FoxWorldContent)
* Личный кабинет с управлением аккаунтом, гильдиями и отображением статистики.
* Систему гильдий - общественных объединений игроков сервера.
* Админ панель с управлением заявками на вступление в сервер (была вырезана в одной из версий, но исходный код можно найти).
* Автоматический деплой при коммитах в main ветку или коммитах в [вики репозитории](https://github.com/PWNight/FoxWorldContent).
* Систему уведомлений о действиях на внутренних системах сайта.
